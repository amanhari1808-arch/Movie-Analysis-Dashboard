# Movie-Analytics-Dashboard (Interactive Dashboard creation using Power BI)
## Project objective
The objective of this project is to analyze movie data using Power BI in order to derive meaningful insights about movie performance, audience preferences, and industry trends. The dashboard is designed to evaluate key performance indicators such as average popularity, total movies, average rating, total revenue, and genre distribution. It provides an interactive platform to explore patterns in movie production, language distribution, and viewer engagement.

## Dataset used
- <a href="https://github.com/amanhari1808-arch/Movie-Analysis-Dashboard/blob/main/Movie%20Dataset.xlsx">Dataset</a>

## Questions (KPIs)
-	What is the average popularity of movies?
-	What is the total number of movies?
-	What is the average duration of movies?
-	What is the average rating across all movies?
-	What is the total revenue generated?
-	Which genres are the most popular?
-	How has movie production grown over the years?
-	Which languages contribute the most to the dataset?
-	Which genres have the highest vote counts?
-	What is the relationship between movie popularity and rating?
-	Which movies have the highest ratings and revenue?
-	Dashboard Interaction <a href="https://github.com/amanhari1808-arch/Movie-Analysis-Dashboard/blob/main/Movie%20dashboard.png">View Dashboard</a>

## Process
- Imported the dataset into Microsoft Power BI Desktop.
- Cleaned the data using Power Query by removing null values, duplicates, and correcting data types.
- Developed the data model and created relationships between relevant fields.
- Created DAX measures for key metrics such as average popularity, total movies, average rating, and total revenue.
- Designed an interactive dashboard using charts, KPI cards, slicers, and tables for better analysis and visualization.

## Project Insight
- Drama, Comedy, and Action are the most popular genres based on overall popularity.
- Movie production has increased significantly over time, especially in recent years.
- English-language movies dominate the dataset, contributing the largest share.
- Certain genres receive higher vote counts, indicating stronger audience engagement.
- Some movies show high popularity but relatively lower ratings, highlighting differences between audience interest and quality perception.
- A small number of movies contribute a large portion of total revenue, indicating the impact of blockbuster films.

## Final Conclusion:
The Power BI dashboard provides a structured and interactive approach to analyzing movie data. It highlights key factors such as genre performance, production trends, and audience preferences.
The insights derived from this analysis can support decision-making in areas such as content production, marketing strategies, and audience targeting. The use of Power BI enables efficient visualization and monitoring of movie-related data, helping to identify trends and improve analytical understanding of the film industry.
